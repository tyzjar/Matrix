## 🎯 Функциональность:

### Что работает:

1. **Переключение вкладок сверху** → меняется раздел параметров
2. **Клик на узел дерева** → открывается подраздел
3. **Разворачивание дерева** → показываются дочерние элементы
4. **Выбор платформы** → переключается активная (зелёная)
5. **Выбор секции** → выделяется активная
6. **Выбор пользователя** → выделяется активный

### Что можно добавить позже:

- Сохранение конфигурации (AJAX запрос)
- Загрузка данных с контроллера (MQTT)
- Валидация полей ввода
- Динамическое добавление/удаление элементов
- Обновление визуализации ГПУ

## 🔧 Как редактировать:

### Изменить цвет:

В `css/config-fullhd.css`:
```css
color: #2A5082;      ← Основной синий
background: #67A979; ← Зелёный активный
```

### Добавить параметр:

В `configuration.html`:
```html
<div class="param_row">
    <label class="param_label">Новый параметр:</label>
    <input type="number" class="input_num" value="0" />
    <span class="param_unit">единица</span>
</div>
```

### Добавить раздел в дерево:

```html
<div class="tree_node">
    <div class="tree_label" data-section="new-section">
        <span class="tree_icon">►</span>
        <span class="tree_text">Новый раздел</span>
    </div>
</div>
```

### Добавить секцию параметров:

```html
<div class="param_section" id="section_new-section" style="display: none;">
    <div class="section_title">
        <h2>Заголовок</h2>
    </div>
    <!-- Параметры -->
</div>
```
