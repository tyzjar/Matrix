## 📋 Что реализовано (по параграфу 8 PDF)

### ✅ Раздел 8.4 - Конструкция ГПУ:

**Список грузоприемных платформ** (Рисунок 8.4.3-1):
- Платформа 1, 2, 3
- Кнопки: ДОБАВИТЬ, УДАЛИТЬ, ОБНОВИТЬ

**Параметры платформы** (Рисунок 8.4.3-2):
- Имя/комментарий
- Количество секций
- Список секций:
  - Секция №1: Длина, Невзвеш. участок
  - Секция №2: Длина, Невзвеш. участок
- Кнопки: ДОБАВИТЬ, УДАЛИТЬ
- Количество опор (датчиков)

**Визуализация ГПУ**:
- Схематичное изображение платформ
- Масштаб по горизонтальной оси
- Обозначения: П1.С1=3,0м и т.д.

### ✅ Раздел 8.6 - Весы (вторая страница):

**Max нагрузка и дискретность**:
- Единицы измерения (г, кг, т)
- Количество диапазонов (1, 2, 3)
- Режим (многоинтервальный, многодиапазонный)
- Max диапазон 1, 2
- Дискретность e=d
- Перегруз

**Калибровка**:
- Калибровать нуль
- Масса груза
- Калибровать нагрузку
- Коррекция нуля

**Параметры фильтра**:
- Тип фильтра (IIR, FIR)
- Частота среза (18Hz, 8Hz, 4Hz, и т.д.)
- Кнопка "Установить"

**Параметры стабильности**:
- Размах колебаний (d)
- Временной интервал (секунд)

## 🔧 Кастомизация

### Изменить цвета:

В `css/configuration-new.css`:
```css
/* Найдите и измените */
color: #2A5082;      ← Основной синий
background: #67A979; ← Зелёный активный
border-color: #B2C2D0; ← Границы
```

### Добавить раздел в дерево:

В `configuration.html`:
```html
<div class="tree_node">
    <div class="tree_label">
        <span class="tree_icon">►</span>
        <span class="tree_text">Новый раздел</span>
    </div>
</div>
```

### Добавить параметр:

```html
<div class="param_row">
    <label class="param_label">Название параметра</label>
    <input type="number" class="param_input_small" value="0" />
    <label>единица</label>
</div>
```
